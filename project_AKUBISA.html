AKUBISA

Project ini adalah project untuk lomba UI/UX IIT Competition 2024 x RumahWeb Indonesia.

Tampilan Project

AKUBISA adalah design UI/UX untuk aplikasi edukasi yang interactive dan tertuju untuk para pelajar agar belajar lebih menyenangkan. AKUBISA memiliki banyak fitur mulai dari materi mata pelajaran, sebuah class dan tugas yang dapat dibuat dan di posting, game quiz menarik, referensi ide, e-book, serta alarm belajar.
