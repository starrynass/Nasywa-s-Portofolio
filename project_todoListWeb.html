To-Do List Bersama Teman

Project to-do list ini adalah project yang saya buat untuk belajar struktur folder laravel.

Tampilan Project

To-Do list ini adalah sebuah web to-do list khusus untuk 2 orang teman, user yang terdaftar dalam database nya pun hanya ada 2 user. fitur pada website ini adalah to-do list bertama, to-do list user ke-1 dan to-do list user ke-2. Dan jika kita ingin membuka dashboard to-do list kita, kita perlu menginput password yang benar. kita juga bisa memasuki to-do list personal teman, jika kita mengetahui password teman kita.

project ini sudah selesai namun belum saya hosting web nya, sehingga belum dapat dibuka ke halaman web perangkat lain.
