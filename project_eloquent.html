DATA KARYAWAN & GAJI

Project ini adalah project untuk tugas sekolah KK3 yaitu pengembangan web, kali ini menggunakan framework laravel dan laravel eloquent.

Tampilan Project

Project ini dibuat dengan menggunakan framework laravel, eloquent dan database mysql. Project ini memiliki fitur untuk menampilkan data karyawan, data gaji, sistem CRUD, fitur search dan relasi antara data karyawan dan data gaji tersebut.
